# 3 Motors Sequential Control using CADe SIMU & Siemens LOGO! Soft

This project demonstrates a simple industrial automation sequence for controlling three motors using Ladder Logic.

## Project Description
The system operates as follows:

1. When the START push button is pressed, Motor 1 (M1) starts immediately.
2. After a 5-second delay, Motor 2 (M2) starts automatically.
3. After another 5-second delay, Motor 3 (M3) starts automatically.
4. The sequence is implemented using timers and ladder logic programming principles.

## Tools Used
- CADe SIMU (Circuit Design and Simulation)
- Siemens LOGO! Soft Comfort
- Ladder Logic Programming

## Learning Objectives
- Understanding sequential motor control.
- Working with timers in ladder logic.
- Simulating industrial automation processes.
- Converting control logic between different PLC simulation environments.

## Applications
- Conveyor systems
- Production lines
- Industrial motor sequencing
- PLC training and education

This project was developed for educational and industrial automation learning purposes.
